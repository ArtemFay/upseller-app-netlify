## Правило архивации

При перемещении модуля в архив:

1. Создавать папку с датой: `<modulename>-YYYY-MM-DD/`.
2. Удалять связанный код из `web/` и `server/server.js`.
3. Обновить запись в этом README.
4. Запись о причине архивации — в коммит-сообщении.

Активный код проекта — `../web/` + `../server/`. См. [`../README.md`](../README.md).
