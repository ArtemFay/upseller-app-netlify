# Archive

Исторические материалы проекта CalendOTG, которые не участвуют напрямую в текущей публикации Netlify.

- `GAS/` - полный бэкап исходной Google Apps Script версии.
- `MIGRATION_PLAN.md` - прежний план миграции и архитектурный контекст.
- `image/` и `calend-otg-*.png` - скриншоты и изображения из миграционного контекста.
- `supabase/` - ранние материалы по Supabase, не подключенные к текущему Netlify-приложению.

Рабочий Netlify-проект находится в `../web/`.
