# CalendOTG_CODEX

Календарь отгрузок Upseller, подготовленный для публикации на Netlify.

## Структура

- `web/` - рабочий Netlify-проект: статические страницы, `netlify.toml`, serverless functions.
- `archive/` - исторический контекст и старые материалы, которые не участвуют напрямую в текущей публикации.

## Как скачать

```powershell
git clone https://github.com/ArtemFay/CalendOTG_CODEX.git
```

Рабочая папка для Netlify: `web/`.
